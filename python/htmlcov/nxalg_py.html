973@mgp.read_proc 

974def weakly_connected_components_subgraph( 

975 vertices: mgp.List[mgp.Vertex], edges: mgp.List[mgp.Edge] 

976) -> mgp.Record(n_components=int, components=mgp.List[mgp.List[mgp.Vertex]]): 

977 """ 

978 This procedure finds weakly connected components of a given subgraph of a 

979 directed graph. 

980 

981 The subgraph is defined by a list of vertices and a list edges which are 

982 passed as arguments of the procedure. More precisely, a set of vertices of 

983 a subgraph contains all vertices provided in a list of vertices along with 

984 all vertices that are endpoints of provided edges. Similarly, a set of 

985 edges of a subgraph contains all edges from the list of provided edges. 

986 

987 The procedure returns 2 fields: 

988 * `n_components` is the number of weakly connected components of the 

989 subgraph. 

990 * `components` is a list of weakly connected components. Each component 

991 is given as a list of `mgp.Vertex` objects from that component. 

992 

993 For example, weakly connected components in a subgraph formed from all 

994 vertices labeled `Person` and edges between such vertices can be obtained 

995 using the following openCypher query: 

996 

997 MATCH (n:Person)-[e]->(m:Person) 

998 WITH collect(n) AS nodes, collect(e) AS edges 

999 CALL wcc.get_components(nodes, edges) YIELD * 

1000 RETURN n_components, components; 

1001 """ 

1002 g = nx.DiGraph() 

1003 g.add_nodes_from(vertices) 

1004 g.add_edges_from([(edge.from_vertex, edge.to_vertex) for edge in edges]) 

1005 

1006 components = [list(wcc) for wcc in nx.weakly_connected_components(g)] 

1007 

1008 return mgp.Record(n_components=len(components), components=components)
